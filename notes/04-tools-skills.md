## Skills 系统

### 概念
- Skill = 可插拔的能力模块
- 按需加载，不使用时不占用资源
- 可以是简单工具或复杂工作流

### 权限管理
- 每个 tool/skill 有权限等级
- 敏感操作需要确认
- 可选沙箱隔离

### MCP 兼容
- 支持 Model Context Protocol
- 可以接入 MCP 服务器提供的工具

## Tool Execution 流程

```
LLM 输出 tool_call
    │
    ▼
权限检查 → 是否允许执行？
    │
    ▼ (允许)
沙箱/直接执行
    │
    ▼
收集结果 → 返回给 Agentic Loop
```
