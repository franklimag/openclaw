# OpenClaw 学习工程 Steering

## 项目目标

本仓库是 OpenClaw 的学习笔记工程，用于系统学习其架构、功能和更新。

## 关键信息

- **学习对象**: OpenClaw — 开源自托管 AI Agent 框架
- **源码仓库**: https://github.com/openclaw/openclaw
- **官方文档**: https://docs.openclaw.ai/
- **技术栈**: JavaScript/TypeScript (Node.js)

## 工作约定

### 目录结构
- `docs/` — 架构文档、参考资料、版本追踪
- `notes/` — 按模块组织的学习笔记
- `experiments/` — 实验代码和配置

### 笔记规范
- 每个模块一个独立 Markdown 文件
- 包含：概念说明、源码位置、问题思考、学习记录
- 使用表格追踪学习进度

### 更新流程
- 发现新版本时更新 `docs/changelog.md`
- 深入研究某模块时更新对应 `notes/` 文件
- 新的参考资料添加到 `docs/references.md`

## 协助要求

当用户询问 OpenClaw 相关问题时：
1. 优先查阅已有笔记是否已覆盖
2. 需要时搜索最新信息并更新笔记
3. 将新发现的知识记录到对应文件中
4. 保持中文撰写
